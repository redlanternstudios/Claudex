# RLS Gamma Formatting Prompt
# Paste before every Gamma deck creation session

---

## CANVAS
- Size: 1920 × 1080px (16:9, Gamma default — do not change)
- Grid: 12 columns, 80px margin, 40px vertical margin
- Safe zone: all readable content within 160px from edges
- Slide rhythm: one clear message per slide

## SLIDE TYPES & USAGE
| Type             | When to use                                |
|------------------|--------------------------------------------|
| Title slide      | Opening only — large display text, minimal |
| Section header   | Separating major deck sections             |
| Content + visual | Main content delivery — text left, image right |
| Full image       | Impact moments — single hero visual        |
| Stats/numbers    | Key metrics — oversized numbers dominant   |
| List/bullets     | 3–5 points max, never more                 |
| Quote            | Single statement, large type, attributed   |
| CTA close        | Final slide — one action, contact info     |

Rule: Maximum 5 bullet points per slide. If more, split the slide.

## TYPOGRAPHY (pixel sizes for Gamma canvas)
| Role         | Size    | Weight |
|--------------|---------|--------|
| Slide title  | 48–64px | 700    |
| Sub-heading  | 28–32px | 600    |
| Body text    | 18–20px | 400    |
| Caption      | 14px    | 400    |
| Stat/number  | 64–96px | 800    |

- Font: Inter
- Heading tracking: tight (-0.02em)
- No text below 14px on slide canvas
- Body line-height: 1.6

## COLORS (RLS Brand — dark default)
- Slide background: #07080D
- Content card / overlay: #111318 with border #2A2D35
- Text: #FFFFFF primary / #9CA3AF secondary
- Accent: #D92532 — section labels, key stats, CTA button
- Light slides (if needed): background #F7F2EE, text #07080D

## VISUAL RULES
- One focal point per slide — eye should know where to go immediately
- Accent red: one use per slide maximum
- Diagrams and icons: flat, monochrome or brand-color only
- No stock photo collages
- White space is not wasted space

## PROMPT STRUCTURE FOR GAMMA (use this order)
```
Topic: [deck topic]
Audience: [who is reading this]
Tone: [executive brief / pitch / technical / educational]
Slides: [approx count, e.g. 10]
Style: dark, minimal, precision brand. Use Inter font. Colors: #07080D background, #FFFFFF text, #D92532 accent.
Structure: [outline if you have one]
```

## ANTI-PATTERNS
- No busy backgrounds behind text
- No more than 2 fonts
- No walls of text (max 60 words of body copy per slide)
- No clip art or cartoon-style icons
- No slide transitions set to "dramatic" or "bounce"
