# /jobpack

Create Rory Semeah's complete application package from one live job URL.

## Usage

`/jobpack <job URL>`

Aliases: `/tailorjob <job URL>`, `tailor resume <job URL>`.

## Required reads

1. Read `.claude/skills/job-application-pack/SKILL.md`.
2. Read `OPS/status/HireWire_Career_Command_Center.md`.
3. Read the current resume and verified career evidence available in the active HireWire workspace.
4. Read the live posting and required official company sources.

## Required output

Deliver all items in one run:

1. One page cover letter PDF and DOCX.
2. One page ATS resume PDF and DOCX.
3. One page company candidate profile PDF and DOCX.
4. Combined three page PDF ordered Cover Letter, Resume, Company Candidate Profile.
5. Fit summary, missing evidence warnings, application questions, and upload guidance.

Do not ask about page order, branding intensity, file names, or whether to complete the whole package. Those decisions are locked. Ask only when the posting cannot be recovered, a required contact or authorization fact is unknown, or an unverified claim would materially alter the application.
