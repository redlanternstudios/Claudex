## Product decision

Do not add more top level features.

Build one compounding loop:

Conversation to private reflection to consented memory to meaningful return.

## Four screen system

### Today

One primary composer and one relevant next action. Every visible control works. Personalization uses verified preferences and activity only.

### History

Searchable conversation titles with open, rename, archive, and delete. Sensitive message previews remain hidden.

### Consent

Selected insight preview, private reflection destination, separate memory permission, exact memory summary preview, and reversible choice.

### Return

One consented memory with provenance, date, reflect action, and guilt free rest action.

## Non negotiables

1. Saving a reflection never enables memory automatically.
2. Memory cannot exist without an active consent receipt.
3. Full chat text never enters memory.
4. Private text never enters analytics.
5. Disabling or deleting memory stops resurfacing immediately.
6. Cross account isolation is tested.
7. Every visible control works.
8. Empty, loading, offline, denied, and retry states exist.
9. Religious guidance remains citation governed.
10. Apple privacy disclosure matches actual behavior.

## Build slices

1. Conversation history.
2. Save insight to private reflection.
3. Memory preview and consent.
4. Consented return card.
5. Privacy safe measurement.
