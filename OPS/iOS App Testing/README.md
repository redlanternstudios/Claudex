## Rule

No iOS build moves to submission until the relevant product testing pack has:

1. Fresh install proof
2. New user proof
3. Existing user proof
4. Paywall proof
5. No payment QA proof
6. RevenueCat restore proof
7. Navigation proof
8. Screenshot proof
9. Known issue list
10. Final go or no go decision
