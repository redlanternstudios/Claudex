## What the owner does

1. Open [PostHog](https://app.posthog.com/signup).
2. Sign in with the dedicated Footprint team email.
3. Create one organization named `By Red, LLC` if the correct organization does not already exist.
4. Create one project named `footprint production`.
5. Invite Ro as owner and Keymon as member.
6. Send Codex only the project region and public project key through the approved secret handoff. Do not paste secret personal values into chat.

## Events Footprint may record

1. `candidate discovered`
2. `candidate qualified`
3. `candidate quarantined`
4. `approval requested`
5. `approval granted`
6. `approval denied`
7. `introduction prepared`
8. `introduction completed`
9. `quality review completed`
10. `stop activated`

## Events Footprint must not record

1. Passwords or tokens
2. Private message bodies
3. Full resumes
4. Sensitive personal data
5. Approval email contents
6. Any data without a clear operating need
