# Session Context Pack

Status: ACTIVE
Owner: Claudex
Purpose: one retrieval path for every Claude and Codex chat in RedLantern Studios

## What this is

This pack is the short list that both engines should read before doing any real work. It keeps the bridge, memory, docs, and vault view in one place so context does not have to be rebuilt from scratch.

## Read order

1. `OPS/BRIDGE.json`
2. `OPS/TODAY.md`
3. `memory/MEMORY.md` index, then only the linked files needed for the task
4. `docs/README.md`
5. `docs/OBSIDIAN_VAULT.md`
6. `OPS/CLAUDEX_SKILL_PROTOCOL.md`
7. `OPS/REDLANTERN_STANDARD_DOCS.md` when the request is document shaped
